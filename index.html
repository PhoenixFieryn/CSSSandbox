CSS Sandbox

Padding and margin, border-box.

Padding determines gap between inner content element width. Margin determines content between outer elements. Border-box property can be set to content-box to make the website elements more consistent. It prevents paddings and borders from adding to the width. The width instead includes the size of the padding and border.

Box model

The box model can be inspected in dev tools - showing various element dimensions. Width/height, padding, border, then margin.

Text Styles

Text can be aligned to various positions relative to their parent element.

This can be useful when creating buttons or other interactive elements.

Text can also be auto indented, making a great start for passages.

Text can also be changed to read backwards. Though this can only be seen if the language used also writes right to left.

The line height for text can be also be changed.

...as well as line spacing...

...and word spacing.

They can be forced to upper or lower case too.

Shadow works too, horizontal vertical blur color.

Relative units

Em, ex, and rem units are relative to the font-size (default 16px) of the document. Em and ex are cumulative to their parent, while rem is independant of their parent. They are more consistent and always scale to the document font-size.

Specificity

Inline styles have a specificity of 1000, followed by ids of 100, classes/attributes/pseudo classes of 10, elements of 1, and universal selectors of 0.
